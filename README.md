## Try it out

Clone the repo

```bash
$ git clone <repo-url>
```

Install dependecies

```bash
$ npm install # or pnpm install or yarn install
```

Build the project

```bash
$ npm build # or pnpm build or yarn build
```

Go to 'about:debugging' on firefox browser

Click 'Load Temporary Add-on'

Select manifest.json file

And that should be it
